# マッシュマッシュ
**自動的**にしいたけを育てる**栽培サービス**です。

## 開発背景
- しいたけを育てるための環境を整えるのが難しい
> 湿度、温度を一定に維持する  
> 水を一定の時間に一定の量をやる
- 実際5人で実験したうえで五つの中三つにかびが生えて育てることが難しいと気づく
